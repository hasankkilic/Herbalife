1️⃣ GitHub Bağlantısı

🔑 Erişim anahtarı (token) nasıl alınır? — tek seferlik kurulum
  1. github.com/settings/personal-access-tokens/new adresine gidin (GitHub hesabınızla).
  2. "Token name" kısmına yorum-paneli yazın; süreyi (Expiration) 1 yıl seçebilirsiniz.
  3. "Repository access" altında Only select repositories → hasankkilic/Herbalife seçin.
  4. "Permissions → Repository permissions → Contents" iznini Read and write yapın.
  5. "Generate token" deyin ve çıkan github_pat_... ile başlayan anahtarı kopyalayıp yukarıya yapıştırın.

⚠️ Anahtar yalnızca bu tarayıcıda saklanır, kimseyle paylaşmayın. Anahtarın ait olduğu hesabın bu repoda yazma yetkisi olmalıdır.
